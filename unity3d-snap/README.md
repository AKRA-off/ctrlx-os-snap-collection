# Unity 3D WebGL Snap 

## Introduction

The sample demonstrates how to connect to pack Unity WebGL application. 

## Function Description

Digital Twin of Rexroth mmS fully controllable form PLC app.

## Implementation Description 

The module __server.py__ contains UDP bridge, HTTP and WebSocket Servers
__snap__ including snapcraft.yaml
__unity-build__ with WebGL build of Unity3D project
__PLC__ program for executing motion in Unity
__setup.py__



## Build

Tested on amd64 devices.
Download the source code and build the snap running command directly:

```shell
snapcraft --destructive-mode
```
